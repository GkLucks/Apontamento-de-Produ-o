<h1>Descrição: </h1>
Sistema acadêmico de controle de apontamentos de produção, desenvolvido para estudo de 
arquitetura, modelagem de domínio, imutabilidade e integração full stack.

## 🔨 Conceitos Técnicos Trabalhados
Imutabilidade de histórico 
● Separação de estado atual vs histórico 
● Regras de domínio no backend 
● Relacionamento N:N 
● Controle de concorrência lógica 
● Soft delete 


## ✔️ Técnicas e tecnologias utilizadas

- ``>NET WEB API``
- ``Entity Framework Core`` ``PostgreSQL``
- ``VUE 3``
- ``Axios``
- ``Pinia``
